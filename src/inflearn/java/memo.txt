* 자바의 실행 방식
  - 소스 코드
    --> 컴파일러를 통한 소스 코드 컴파일(javac --> (Hello.java -> Hello.class))
    --> JVM 실행및 프로그램 실행

  - 자바는 OS에 자유롭다.
    --> JVM이 OS에 따른 명령어를 만들어 준다.

* 변수
  - 값을 넣을 수 있는 상자.
  - 변수는 언제든지 값을 바꿀 수 있다.(값을 바꾸는 경우 기존의 값은 삭제된다.)
  - int a = 10; --> 정수형 변수 a를 선언 및 초기화(값을 넣어주는 것).
  - 변수 선언 -> 컴퓨터 메모리에 공간을 확보해서 그곳에 데이터를 저장 + 이름(변수명) 부여
  - 변수 초기화 -> 변수 선언 후 처음으로 값을 넣어주는 것.
  - 변수를 사용하려면 반드시..! 초기화를 해줘야한다. (컴파일 오류)
    --> 메모리 상에 어떤 값이 들어 있는지 알 수가 없기 때문에 이상한 값(버그)를 방지하기 위해서 컴파일 오류가 난다.
  - 리터럴(literal)은 문자, 글자를 의미한다. 값 자체가 리터럴 이라고 할 수 있다.
